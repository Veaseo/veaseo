« Je veux juste un site simple, une seule page, pas compliqué. »

Cette phrase, je l'entends 1 fois sur 3 lors des briefs clients. Et 8 fois sur 10, c'est un **mauvais choix** pour leur business. Voici pourquoi — et quand, à l'inverse, le one-page est la meilleure option.

## C'est quoi un site one-page ?

Un **one-page** (ou single page) = un site composé d'**une seule page web**, avec différentes sections scrollables (Accueil, Services, Témoignages, Contact, etc.).

Tout est sur la même URL : `votreentreprise.fr`.

Un **multi-pages** = un site avec plusieurs pages distinctes : `/`, `/services`, `/a-propos`, `/contact`, etc. Chaque page a sa propre URL et son propre H1.

## Les vraies différences (au-delà du marketing)

### 1. Le SEO : multi-pages gagne (très largement)

Pour Google, **chaque URL est une opportunité de ranker** sur un mot-clé spécifique.

Avec un one-page :
- 1 seule URL → 1 H1 → 1 thématique principale
- Vous pouvez ranker sur **1 ou 2 mots-clés maximum**
- Impossible de cibler plusieurs villes ou plusieurs services

Avec un multi-pages :
- 10-20 URLs distinctes → 10-20 opportunités de ranker
- Une page par service = 1 mot-clé principal par page
- Une page par ville = SEO local sur plusieurs zones
- Le **blog** ajoute encore plus d'opportunités

→ **Pour le SEO local**, le one-page est presque toujours un mauvais choix.

### 2. La conversion : multi-pages gagne aussi

Un visiteur qui cherche « tarifs plomberie » ne veut pas scroller votre histoire avant de trouver les tarifs.

Avec un multi-pages :
- Le visiteur **arrive directement** sur la page qui l'intéresse depuis Google
- Le **parcours de conversion** est ciblé par intention
- Vous pouvez **tracker** plus finement quelle page convertit

Avec un one-page :
- Le visiteur doit **scroller / chercher** la section qui l'intéresse
- Pas d'analytics fines page par page
- Risque de rebond avant d'atteindre l'info pertinente

### 3. Le coût : variable selon la complexité

| | One-page | Multi-pages |
|---|---|---|
| Site simple | **800 - 2 000€** | 3 000 - 5 000€ |
| Site complexe | 2 000 - 4 000€ | 5 000 - 10 000€ |

Le one-page **n'est pas toujours moins cher** : un one-page très long (long-form sales page) peut coûter plus qu'un multi-pages basique.

### 4. La maintenance et l'évolution

Un site multi-pages est **plus évolutif** :
- Vous voulez ajouter un service ? Nouvelle page, sans toucher au reste
- Vous voulez ajouter un blog ? Pas de souci
- Vous voulez cibler une nouvelle ville ? Nouvelle page locale

Un one-page devient vite **rigide** : chaque évolution demande de retoucher la structure entière.

## Quand le one-page est le BON choix

Soyons honnêtes : il y a des cas où le one-page est pertinent.

### Cas 1 : Landing page de campagne ponctuelle

Vous lancez un événement, une formation, un produit one-shot. **Pas besoin de multi-pages**, vous voulez convertir sur 1 objectif unique. One-page focalisé = bon choix.

### Cas 2 : Activité ultra-simple, 1 service, 1 zone

Vous êtes coach de yoga, vous donnez des cours dans 1 ville, vous n'avez qu'1 type de service. Un one-page peut suffire.

⚠️ Mais même là, vous limitez votre SEO local.

### Cas 3 : Carte de visite digitale (faible enjeu SEO)

Vous êtes consultant indépendant qui n'a pas besoin de Google pour trouver des clients (réseau, recommandations). Vous voulez juste un site « présence » pour rassurer les prospects qui googlent votre nom. One-page suffit.

### Cas 4 : Prototype / MVP avant projet plus ambitieux

Vous lancez une activité et voulez tester avant d'investir. One-page **temporaire** = OK.

## Quand le multi-pages s'impose

Pour 90% des entreprises locales avec un vrai enjeu commercial, le multi-pages est non-négociable. Notamment :

- ✅ **Vous voulez ranker sur Google** sur plusieurs mots-clés / villes
- ✅ Vous avez **plusieurs services** distincts
- ✅ Vous voulez **un blog** SEO actif
- ✅ Vous intervenez dans **plusieurs zones géographiques**
- ✅ Vous avez **plusieurs cibles** de clients
- ✅ Vous voulez un **espace presse / mentions / témoignages** détaillés
- ✅ Vous prévoyez de **faire grandir** votre activité

## La structure multi-pages minimale pour entreprise locale

Si vous partez sur un multi-pages, voici la structure minimale recommandée :

1. **Accueil** (`/`) — vitrine + positionnement
2. **Services** (`/services`) — hub + page par service
3. **Réalisations** (`/realisations`) — preuves sociales
4. **À propos** (`/a-propos`) — confiance / storytelling
5. **Blog** (`/blog`) — SEO + autorité
6. **Contact** (`/contact`) — formulaire + coordonnées
7. **Pages locales** (`/zones-intervention/[ville]`) — SEO local
8. **Mentions légales** (`/mentions-legales`) — obligatoire

C'est la structure que j'utilise sur tous mes projets, y compris ce site.
